## Storybook in Next.js

This repo shows that storybook can apply on Next.js .

### How To Run Storybook

```sh
$ cd NEXT-STORYBOOK
$ npm run storybook
```

### How to Run Apps
```sh
$ cd NEXT-STORYBOOK
$ npm run dev
```
